# Oracle-Demo
Demo Oracle PL/SQL
[Read Me.pdf](https://github.com/user-attachments/files/26331604/Read.Me.pdf)
A demo Oracle application built with the PL/SQL programming language.
The schema is based on the famous Scott/Tiger training database, with additional tables for importing data, calculating statistics, and
defining holiday dates.
Sample code includes: Error logging, CSV record handling and field extract, Import/Export of CSV data, Numeric, Date, and String functions,
Binary Search, Sort, Remove Duplicates, Statistics.
